Stap 1 — GitHub repository aanmaken

1. Ga naar https://github.com/lon-ka → New repository
2. Naam: KA-Schooluitstappen | Visibility: Public | Add README: ✓
3. Hernoem KA-Schooluitstappen.html naar index.html en push naar de main branch
4. Ga naar Settings → Pages → Source: main branch / root → Save
5. Wacht ~2 minuten, daarna bereikbaar op https://lon-ka.github.io/KA-Schooluitstappen/

Stap 2 — Azure AD app-registratie bijwerken

1. Ga naar https://entra.microsoft.com → App-registraties → Schooluitstappen
2. Verificatie → + URI toevoegen (onder SPA-platform)
3. Voeg toe: https://lon-ka.github.io/KA-Schooluitstappen/
4. Klik Opslaan

Stap 4 — Gebruikers toevoegen

Voeg elke gebruiker toe in de Gebruikers-lijst:

Title:  voornaam.naam@gebaska.be   (exact Microsoft 365-adres)
School: 't Mulderke                 (of Het Kiezeltje / De Kei)
Rol:    leerkracht                  (of secretariaat / directie)
⚠️ Leerkracht op meerdere scholen? Voeg meerdere rijen toe — één per school.

Stap 5 — Testen

1. Ga naar https://lon-ka.github.io/KA-Schooluitstappen/
2. Klik Aanmelden met Microsoft 365
3. Meld aan met je schoolaccount
4. Dashboard moet openen met je school en rol rechtsboven

Instellingen na eerste aanmelding

Ga als directie of secretariaat naar ⚙️ Instellingen en vul in:
• Noodnummers (directie + secretariaat)
• Klasnamen per school worden automatisch geladen vanuit de standaardconfiguratie
